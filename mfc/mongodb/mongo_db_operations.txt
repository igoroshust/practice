В MongoDB существует множество операторов, которые можно использовать для выполнения различных операций с документами в коллекциях. Вот список основных операторов, сгруппированных по категориям:

1. Операторы сравнения
$eq: равно (equal)
$ne: не равно (not equal)
$gt: больше (greater than)
$gte: больше или равно (greater than or equal)
$lt: меньше (less than)
$lte: меньше или равно (less than or equal)
$in: находится в списке значений (in)
$nin: не находится в списке значений (not in)

2. Логические операторы
$and: логическое "И" (and)
$or: логическое "ИЛИ" (or)
$not: логическое "НЕ" (not)
$nor: логическое "НИ" (nor)

3. Операторы для работы с массивами
$all: соответствует всем элементам массива (all)
$elemMatch: соответствует хотя бы одному элементу массива, который соответствует заданным условиям (elemMatch)
$size: соответствует массивам определенного размера (size)

4. Операторы для работы с регулярными выражениями
$regex: соответствует строкам, которые соответствуют регулярному выражению (regex)
$options: задает параметры для регулярных выражений (например, игнорирование регистра)

5. Операторы для работы с полями
$exists: проверяет, существует ли поле (exists)
$type: проверяет тип поля (type)

6. Операторы для обновления
$set: устанавливает значение поля (set)
$unset: удаляет поле (unset)
$inc: увеличивает значение поля (increment)
$push: добавляет элемент в массив (push)
$addToSet: добавляет элемент в массив, если его там нет (addToSet)
$pop: удаляет первый или последний элемент массива (pop)
$pull: удаляет все элементы из массива, которые соответствуют условию (pull)
$rename: переименовывает поле (rename)

7. Операторы агрегации
$group: группирует документы по определенному полю (group)
$match: фильтрует документы (match)
$project: изменяет структуру документов (project)
$sort: сортирует документы (sort)
$limit: ограничивает количество возвращаемых документов (limit)
$skip: пропускает указанное количество документов (skip)

8. Специальные операторы
$currentDate: устанавливает поле на текущее время (currentDate)
$merge: объединяет результаты агрегации с коллекцией (merge)
$out: записывает результаты агрегации в новую коллекцию (out)
